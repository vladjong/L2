Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:
```
[77, 78, 79]

Так-как мы делаем слайс от [1;4). Перед этим копируем в слайс b[] массив a[5].

```
